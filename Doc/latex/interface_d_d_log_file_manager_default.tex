\subsection{Detailed Description}
Default log file manager.

All log files are placed inside the logs\-Directory. If a specific logs\-Directory isn't specified, the default directory is used. On Mac, this is in $\sim$/\-Library/\-Logs/$<$\-Application name$>$=\char`\"{}\char`\"{}$>$. On i\-Phone, this is in $\sim$/\-Library/\-Caches/\-Logs.

Log files are named \char`\"{}log-\/$<$uuid$>$.\-txt\char`\"{}, where uuid is a 6 character hexadecimal consisting of the set \mbox{[}0123456789\-A\-B\-C\-D\-E\-F\mbox{]}.

Archived log files are automatically deleted according to the maximum\-Number\-Of\-Log\-Files property.
